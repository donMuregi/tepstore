Excel File Format

The Excel file should have the following columns in order:

  1. Name (required) - Employer/Company name
  2. Code - Unique employer code
  3. Address - Company address
  4. Contact Person - HR or contact person name
  5. Contact Email - Contact email address
  6. Contact Phone - Contact phone number

The first row should be headers and will be skipped.
