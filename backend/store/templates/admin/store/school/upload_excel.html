Excel File Format

The Excel file should have the following columns in order:

  1. Name (required) - School name
  2. Location - City or area (e.g., Nairobi, Mombasa)
  3. County - County name (e.g., Nairobi, Kiambu)
  4. School Type - Type of school (e.g., Primary, Secondary, Mixed)

The first row should be headers and will be skipped.

Note: If a school with the same name already exists, it will be updated with the new information.
